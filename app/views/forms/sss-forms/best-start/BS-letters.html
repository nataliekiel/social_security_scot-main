Letters in other formats

We'll send you a letter to let you know whether you're going to be paid Best Start Grant and Best Start Foods. This letter will be in English. You can ask for the letter in another language if needed later.

If you're not able to read a letter in normal print, we can also send you an extra letter as an audio file or in:

  • braille
  • easy read
  • large print
